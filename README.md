# News Explorer iOS App

A UIKit-based News Explorer application built using **MVVM-C architecture** and **RxSwift**.  
The app fetches and displays news articles from NewsAPI with a clean, scalable, and maintainable codebase.

## Tech Stack
- UIKit (Programmatic UI)
- RxSwift / RxCocoa
- MVVM-C Architecture
- Repository Pattern
- URLSession (no third-party networking or image libraries)

## Features
- News list using UITableView
- Asynchronous API calls with error handling
- Clean separation of concerns
- iPhone-first, iPad compatible

## Estimated Development Time
**8–10 hours**
- Project setup & architecture: 2h  
- Network & repository layer: 2h  
- ViewModel & Rx bindings: 2h  
- UI & table view setup: 2h  
- Testing, cleanup & polish: 1–2h
